Widgets Toolbox

Widgets Toolbox is a collection of higher-level user interface components, building blocks, and tools for developing complex and professional applications in MATLAB.
These widgets combine existing control functionalities together into something larger, implementing a common functionality and logic that is useful in multiple places throughout graphical user interfaces.
This toolbox was developed by Robyn Jackey and others from the Consulting Services team at MathWorks.
